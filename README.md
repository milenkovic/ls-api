# API service

## Prerequisites

Node.js https://nodejs.org/en/download/ (v10.16.3)

## Getting started

```bash
$ npm i
$ npm run build
$ npm start
```
App will be available on port 3333

Port can be defined in env/ENVIRONMENT_NAME.env file

## Tests

```bash
$ npm run test
```

## Lint

```bash
$ npm run test
```

## Run in Dev mode

```bash
$ npm run dev
```
